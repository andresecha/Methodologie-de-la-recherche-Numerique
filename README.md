## 🇫🇷 Français

Ce dépôt rassemble des guides méthodologiques et des environnements de travail basés sur des carnets Jupyter (Google Colab) conçus pour les chercheurs en **Sciences Humaines et Sociales (SHS)**. L'objectif est de faciliter la transformation des sources primaires (documents imprimés, manuscrits et fichiers sonores) en données textuelles structurées et exploitables.

Ce projet est développé dans le cadre du **Groupe d'intérêt sur l'acquisition assistée de textes** du **Consortium Huma-Num ARIANE**.

### 🛠️ Outils Disponibles

| Outil | Description | Accès |
| :--- | :--- | :--- |
| **[OCR avec Mistral AI](./OCR/Mistral/)** | Extraction de texte, tableaux et LaTeX à partir de PDFs complexes et manuscrits. | [Guide FR](./OCR/Mistral/Metodological_guide_MistralOCR_Transcripcion_ARIANE_FR.ipynb) |
| **[Speech-to-Text avec Whisper](./Speech-to-text/Whisper/)** | Transcription et traduction automatique de l'audio (entretiens, archives sonores). | [Guide FR](./Speech-to-text/Whisper/Metodological_guide_Whisper_audio_Transcripcion_ARIANE_FR.ipynb) |
